## A Guide to the Entry Metadata:

**Museum Entry Link**: Extremely helpful links to origin repositories/other databases that have publicly uploaded the same entry online. Here you can gather significant amounts of additional metadata, often straight from museums and credible actors. 

**Accession Number**: The direct identification number the origin repository uses to identify the depicted item on their database. Provided occasionally but not always. 

**Subjects**: "Tags" that you can sort the database by.

**Earliest Date and Latest Date**: The most probable time period/year we could find for when an item was actually created, or at the very least obtained by a museum. Often a span of time, i.e. earliest date=750BC and latest date = 690BC. If vague, try checking repository entry link if there is one.

**Grouping**: Notes if entry belongs to a collection or if there are multiple photos of the same object depicting additional angles.

**Collection Notes**: Distinguishes what collection/ set of alternate angles an item belongs to, if it has a grouping. This means that there are multiple database pictures of this same exhibit focusing on different items pictured, i.e. two clarinets in one photo will likely have two unique entries, in which both can be found by searching this collection grouping name.

**Identifier**: Unique string of numbers associated with only one entry. You can find this in the url of any entry. 

**Relationships**: Occasionally entries in collections/alternate angles will also have this category, which gives the name of these related files. One could then directly find related entries by searching the given filename on the database.
